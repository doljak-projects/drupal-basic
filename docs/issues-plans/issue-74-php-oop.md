# Issue #74 — PHP Object-Oriented Programming — Coursera course

## Objective
Dedicated study of PHP Object-Oriented Programming through the Coursera course.
This is a standalone PHP exercise track — not tied to Drupal — focused on
building a solid OOP foundation that will support the advanced Drupal topics
already in scope.

## Scope
- Study and apply core OOP concepts: classes, objects, properties, methods
- Practice inheritance, interfaces, abstract classes and traits
- Explore visibility modifiers (public, protected, private)
- Model entities as PHP classes (identity, state, behavior separation)
- Apply basic caching patterns in PHP (memoization, object cache)
- Implement course exercises as standalone PHP scripts
- Document key concepts and decisions as the course progresses
